# Contributing to AI-Pedagogy

Thank you for your interest in contributing to AIP (Artificial Intelligence Pedagogy)!

## About This Project

This is a research prototype developed for academic purposes. The project explores pedagogical interactions between artificial agents using Large Language Models.

## Current Status

This repository is primarily maintained for research purposes and reproducibility of the findings published in:

- **Patania et al. (2025)** – *AI Pedagogy: Dialogic Social Learning for Artificial Agents*, ICSR 2025.
- **Patania et al. (2026)** – *Teaching LLMs Naturally: Pedagogical Strategies for Interactive Knowledge Acquisition*, accepted at AAMAS 2026.

## How to Use This Code

You are welcome to:
- Fork the repository for your own research
- Experiment with different configurations
- Adapt the code for related research questions
- Report bugs or issues

## Development Guidelines

If you do wish to contribute:

1. **Fork the repository** and create a new branch for your changes
2. **Follow existing code style** and conventions
3. **Write clear commit messages** explaining your changes
4. **Test your changes** before submitting
5. **Submit a pull request** with a clear description

## Reporting Issues

If you encounter bugs or have questions:
- Open an issue with a clear description
- Include relevant logs and configuration details
- Specify your environment (Python version, OS, etc.)

## Contact

For research collaborations or questions about the methodology, please contact the authors through the paper's contact information.

**Note**: This is a research project and contributions will be reviewed based on their alignment with the research goals.
